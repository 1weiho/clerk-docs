# `<MultisessionAppSupport>`
---
The `<MultisessionAppSupport>` provides a wrapper for your React application that guarantees a full rerendering cycle everytime the current session and user changes. The `<MultisessionAppSupport>` should live under the `<ClerkProvider/>` and wrap your React application.

```jsx copy filename='app.js'
import { useEffect } from "react";
import { ClerkProvider, MultisessionAppSupport } from "@clerk/clerk-react";

function App() {
  return (
    <ClerkProvider publishableKey="pk_test_••••••••••••••••••••••••••••••••••">
      <MultisessionAppSupport>
        <Page />  
      </MultisessionAppSupport>
    </ClerkProvider>
  );
}

function Page() {
  useEffect(() => {
    // No need to check if the 
    // Clerk object exists.
    document.title = "This page uses Clerk " + 
      document.window.Clerk.version;
  }, []);
  
  return (
    <div>The content</div>
  );
}
```
