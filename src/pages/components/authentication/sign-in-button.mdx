# `<SignInButton />`

`<SignInButton/>` is a button that links to the sign-in page or displays the sign-in modal.

## Usage

<Tabs items={['Next.js', 'React', 'Remix', 'Gatsby']}>
<Tab>
```jsx copy filename="pages/index.js"
import { SignInButton } from "@clerk/nextjs";

export default function Home() {
  return (
    <div>
    <h1> Sign in </h1>
      <SignInButton />
    </div>
  );
}
```
You can also overide the default button with your own button and text.

```jsx copy filename="pages/index.js"
import { SignInButton } from "@clerk/nextjs";

export default function Home() {
  return (
    <div>
      <h1> Sign in </h1>
      <SignInButton mode="modal">
        <button className="btn">Sign in</button>
      </SignInButton>
    </div>
  );
}
```

</Tab>
