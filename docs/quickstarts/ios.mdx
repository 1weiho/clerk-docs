### Install the SDK

Set up the Clerk IOS SDK in Xcode using [Swift Package Manager](https://developer.apple.com/documentation/xcode/adding-package-dependencies-to-your-app):

1. In Xcode, navigate to **File > Add Package Dependencies**.
1. In the **Search or Enter Package URL** field, enter the following repository URL:
    ```txt filename="IOS SDK Repo URL"
    https://github.com/clerk/clerk-ios
    ```
1. Then, select your preferred dependency rule and select **Add Package**.

### Configure and load Clerk

To configure Clerk in your local project, you first need your publishable API key:

<SignedIn>

<Callout type="info">
  Because you're signed in, the code snippet below will contain the publishable key for your most recently active project. Unless you want to use a different project, you can skip these steps.
</Callout>

</SignedIn>

1. Navigate to the Clerk Dashboard.
2. In the navigation sidebar, select [API Keys](https://dashboard.clerk.com/last-active?path=api-keys).
3. In the **Quick Copy** section, copy your Clerk publishable key.
4. Add the following code snippet to your project, assigning your publishable key to the `publishableKey` property:

### Create a sign-in view

To see Clerk in action, create a view that allows visitors to sign-in to your Clerk app:

<InjectKeys>

<Tabs type="framework" items={["SwiftUI", "UIKit"]}>

<Tab>
<CodeBlockTabs options={["SignInView.swift"]}>

```swift filename="SignInView.swift"
import SwiftUI
import ClerkSDK

struct SignInView: View {
    @ObservedObject private var clerk = Clerk.shared
    @State private var emailAddress: String = ""
    @State private var password: String = ""
    
    var body: some View {
        VStack {
            if let user = clerk.user {
                Text(user.id)
            } else {
                TextField("Email", text: $emailAddress)
                SecureField("Password", text: $password)
                Button("Sign In", action: {
                    Task {
                        try? await clerk.client?.createSignIn(strategy: .identifier(emailAddress, password: password))
                    }
                })
            }
        }
    }
}
```

</CodeBlockTabs>
</Tab>

<Tab>
<CodeBlockTabs options={["SignInViewController.swift"]}>
```swift
import UIKit
import ClerkSDK

class SignInViewController: UIViewController {
    @IBOutlet weak var userIdLabel: UILabel!
    @IBOutlet weak var emailTextField: UITextField!
    @IBOutlet weak var passwordTextField: UITextField!
        
    @IBAction func signIn(_ sender: Any) {
        guard let emailAddress = emailTextField.text, let password = passwordTextField.text else {
            return
        }
        
        Task {
            try? await Clerk.shared.client?.createSignIn(strategy: .identifier(emailAddress, password: password))
            userIdLabel.text = Clerk.shared.user?.id
        }
    }
}
```
