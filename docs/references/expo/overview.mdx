# Expo and Clerk

<Callout type="warning">
  Currently the `@clerk/expo` SDK only supports mobile app output. You can upvote the addition of Clerk supporting web output [here](https://feedback.clerk.com/roadmap/fb56a8db-763a-4e0d-a4b2-f8a453be195c).
</Callout>

Clerk makes it simple to add authentication to your Expo application. This documentation covers the capabilities and methods available from Clerk's Expo SDK.

See [the quickstart](/docs/quickstarts/expo) to get started.
